# ScottCom
A small PECS board for my son

Electronic Part List:

1x Arduino Pro Mini 3.3V 8Mhz
1x DF Mini Player
1x Micro SD Card
1x TP 4056
1x Speaker 8Ohm 40mm diameter
1x Battery 3.7V 1700mAh (can be more or less Amps)
1x Micro USB Breakout Board
1x Micro Switch
1x 5mm Led
1x 220R Resistor R3
1x 12k Resistor R2
1x 33k Resistor R1

Craft Part List
3mm MDF
10x M3 10mm srews
